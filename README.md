# DQ2 シドー戦バトルゲーム

ドラゴンクエストII風のターン制バトルゲームです。

## 🎮 プレイ方法

GitHub Pages: https://big-blue22.github.io/DQ/

## 🚀 ローカル開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## 📦 技術スタック

- React
- Vite
- Lucide React (アイコン)
- CSS

## 🎯 ゲームの特徴

- 3人のパーティー（ローレシア、サマルトリア、ムーンブルク）
- 破壊神シドーとの戦い
- 攻撃、呪文、防御のコマンド
- ベギラマ、イオナズン、メラゾーマなどの呪文
- 回復・蘇生システム
- レトロなドラクエ風UI
