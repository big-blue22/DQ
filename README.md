# DQ2 シドー戦バトルゲーム

ドラゴンクエストII風のターン制バトルゲームです。

## 🎮 プレイ方法

GitHub Pages: https://big-blue22.github.io/DQ/

## 🚀 ローカル開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## 📦 技術スタック

- React
- Vite
- Lucide React (アイコン)
- CSS

## 🎯 ゲームの特徴

- 3人のパーティー（ローレシア、サマルトリア、ムーンブルク）
- 破壊神シドーとの戦い
- **レトロJRPG風UI** - DQ2の画面デザインを忠実に再現
- **新コマンドシステム**:
  - 「こうげき」「ぼうぎょ」「どうぐ」「にげる」の4つのコマンド
  - 「こうげき」選択時に「ぶきで こうげき」「じゅもん」のサブメニュー
  - キャラクター別の呪文使用制御（ローレシアは呪文不可）
- レベル、HP、MPを表示するパーティステータスウィンドウ
- イオナズン、メラゾーマ、ベホマなどの呪文
- 回復・蘇生システム
- 敵スプライト表示
